	動作環境について
________________________

	※「Grand Theft Auto: Vice City」（以下、GTA:ViceCity）をプレイするには、
	Microsoft(R) DirectX(R) 8.1以上が必要です（ゲームのCD-ROMにはDirectX(R) 8.1が収録されています）。

	※ご使用のOSがMicrosoft(R) Windows(R) 98及び98SEであり、CPUがIntel(R) Pentium(R)4である場合、
	動作に不具合が生じる場合がございます。ご注意ください。

必須動作環境
	●OS：Microsoft(R) Windows(R) 98/98SE/Me/2000/XP
	●CPU：Intel(R) Pentium(R)III 800MHz以上
	●メモリ：128MB以上
	●HDD空き容量：1.6GB以上
	●CD-ROMドライブ：4倍速以上
	●モニタ：SVGA（640×480）16ビットカラー以上
	●ビデオカード：Direct3D対応AGPビデオカード（DirectX(R) 8.1以上）VRAM32MB以上 
	※システムとメモリを共有するタイプのものは、動作保証外となります。
	●サウンドカード：DirectSound対応サウンドカード（DirectX(R) 8.1以上）（EAX対応）
	※オンボードでの動作は保証対象外となります。
	●入力デバイス：マウス、キーボード（ゲームパッド対応）

推奨動作環境
	●OS：Microsoft(R) Windows(R) 2000/XP
	●CPU：Intel(R) Pentium(R)4 2GHz以上
	●メモリ：256MB以上
	●HD空き容量：1.6GB以上
	●CD-ROMドライブ：32倍速以上
	●モニタ：SVGA（640×480）16ビットカラー以上
	●ビデオカード：Direct3D対応AGPビデオカード(DirectX(R)8.1以上)VRAM64MB以上 
	※システムとメモリを共有するタイプのものは、動作保証外となります。
	●サウンドカード：DirectSound対応サウンドカード（DirectX(R)8.1以上）EAX対応 
	※オンボードでの動作は保証対象外となります。
	●入力デバイス：マウス、キーボード（ゲームパッド対応）

	※ご使用のOSがMicrosoft(R) Windows(R) 2000の場合、Service Pack 3以降が必要です。
	※ご使用のOSがMicrosoft(R) Windows(R) XPの場合、Service Pack 1以降が必要です。
	※以下のOSはサポートしていません。
		Microsoft(R) Windows(R) 95/NT（バージョンを問いません。）
	※お客様ご使用のパソコンが、DirectX(R) 8.1以上に対応されているかを必ずご確認ください。
	※最新のディスプレイドライバ、サウンドドライバをあらかじめご用意いただかなくてはならない場合がございます。
	※自作機、改造機は動作保証外とさせていただきます。あらかじめご容赦ください。

	注意事項
_________________

日本語版以外の「GTA:ViceCity」をインストールされていた方へ：
ご使用のコンピュータに日本語版以外の「GTA:ViceCity」をインストールされていた場合、
ゲームが作成したファイルやフォルダが残されている可能性があります。
また、[マイ ドキュメント]内の[GTA Vice City User Files]フォルダに、
ゲームのセーブデータや各種の設定ファイルなどが残されている場合もあります。
これらのファイルやフォルダを残したまま「GTA:ViceCity」日本語版をインストールされますと、
ゲームに不具合が生じる可能性がありますので、インストール前に削除することを強くお勧めいたします。
また、これらのファイルやフォルダを保存する必要がある場合は、
あらかじめ他のフォルダやディスクなどに移動しておいてください。
「GTA:ViceCity」を初めてインストールされる場合は、こうした作業をしていただく必要はありません。

インストール途中の「コンポーネントの選択」について：
ゲーム中で車などに乗った際に流れるBGM（ラジオ局）のデータをハードディスクにインストールします。
通常はインストールすることによってゲームのパフォーマンスが向上しますが、
速度の遅いPCの場合は、逆に低下する場合もありますのでご注意ください。

セーブデータについて：
Microsoft(R) Windows(R) 2000/XPをお使いの場合、ログイン中のユーザーの[マイ ドキュメント]内の
[GTA Vice City User Files]フォルダに、ゲームのセーブデータ、各種の設定ファイルなどが保存されます。
ゲームの各種データのバックアップを取っておきたい時にはこのフォルダをご参照ください。

アンインストールの際の注意点：
「GTA:ViceCity」をアンインストールする際、ゲームのインストールフォルダにある[mp3]と[skins]フォルダ、
[マイ ドキュメント]内の[GTA Vice City User Files]フォルダは削除されません。
これらのフォルダとファイルの削除は、各ユーザーの判断に基づいておこなってください。


「フレーム制限」について：
「GTA:ViceCity」の「ディスプレイ設定」で「フレーム制限」を無効にすることはお奨めしません。
この設定を無効にすると、ゲームのプレイ速度のばらつきが大きくなってしまいます。
